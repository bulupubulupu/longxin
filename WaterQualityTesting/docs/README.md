此项目是基于龙芯2k500的智慧水质检测系统

项目开始时间 2024年4月

项目结构包括

├── src/             # 源代码目录

├── include/         # 头文件目录

├── lib/             # 库文件目录（如果项目使用或创建库）

├── bin/             # 编译后的可执行文件目录

├── obj/             # 编译过程中生成的中间文件（如.o文件）

├── tests/           # 测试代码目录

├── docs/            # 文档目录

├── scripts/         # 脚本目录（如安装脚本、配置脚本等）

├── Makefile         # 构建项目使用的Makefile

└── CMakeLists.txt   # 如果使用CMake构建系统
